# Branding Assets

## Logo de l'entreprise

Placez le logo de votre entreprise dans ce dossier avec le nom `logo.png` ou `logo.svg`.

### Formats acceptés
- PNG (recommandé pour fond transparent)
- SVG (idéal pour le scaling)
- JPG (si pas de transparence)

### Dimensions recommandées
- Logo header : 200-300px de largeur
- Logo page d'accueil : 400-600px de largeur
- Format : Transparent ou fond blanc/noir selon le design

### Nom des fichiers
- `logo.png` ou `logo.svg` : Logo principal
- `logo-white.png` : Version blanche (optionnel)
- `logo-small.png` : Version petite pour header (optionnel)
